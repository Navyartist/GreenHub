import tkinter as tk
# tkinter 쓸건데 이름이 너무 길다. 내맘대로 tk란 이름으로 쓰겠다.

from tkinter import *
# tkinter 안에 있는 모든 것(*)을 여기로 가져와 쓰겠다.

from tkinter import ttk
# tkinter에서 확장된 ttk란 모듈을 가져와서 쓰겠다. 다른건 못 씀

ttk는 기본 tkinter에서 위젯등을 더 예쁘고 세련되게 볼수있게 기능을 좀더 확장한 모듈이다.
3번째 줄(only ttk)만 작성해놓으면 기본 tkinter에 있는 클래스들은 못 쓴다.

방식 | 위젯 생성 예시 | 특징
import tkinter as tk | tk.Label(...) | 가장 안전함. 출처가 명확함.
from tkinter import * | Label(...) | 쓰기는 편하지만 위험함. (이름 중복 가능성)
from tkinter import ttk | ttk.Label(...) | 기본형보다 예쁜(현대적인) 위젯 사용 시 필요.
